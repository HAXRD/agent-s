# 加密货币实时数据监控系统 - Spec Shaping

**创建日期：** 2025-11-07  
**状态：** ✅ 需求收集完成

## 功能概述

基于产品规划文档，实现一个个人使用的加密货币交易对实时数据监控系统，包括：
- 实时K线图展示（可交互，包含成交量图）
- 代币对选择（下拉选择框，默认BTC/USDT）
- 时间粒度选择（按钮组：1m, 5m, 1h）
- 实时数据WebSocket推送（节流更新）
- 实时数据自动落表存储（所有时间粒度）

## 技术栈

- **后端：** Go 1.25.3 + Gin + GORM + PostgreSQL
- **前端：** React 18+ + Vite + TradingView Lightweight Charts
- **数据源：** Binance Public API
- **实时通信：** WebSocket (Gorilla WebSocket)

## 关键决策

### 数据存储策略
- **存储粒度：** 存储所有时间粒度的K线数据（1m, 5m, 1h分别存储）
- **数据保留：** 永久保存（个人使用，数据量可控）
- **历史查询：** 查询所有历史数据

### 界面布局
- **布局：** 全屏K线图，控制栏在顶部
- **价格显示：** 右侧Y轴上显示当前价格（红色标签）
- **成交量：** K线图下方显示成交量柱状图

### 交互细节
- **悬浮信息：** 显示价格、开盘价、最高价、最低价、收盘价、成交量、时间戳
- **价格标签：** 十字线跟随鼠标时在图表上显示价格标签
- **错误处理：** 显示错误提示 + 静默重试 + 记录日志
